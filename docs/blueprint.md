# **App Name**: Rohit Mirge Data Portfolio

## Core Features:

- Profile Showcase: Display a professional profile highlighting Rohit Mirge's skills and experience as a data analyst, populated by data scraped from LinkedIn.
- Project Portfolio: Showcase data analysis projects with descriptions, visualizations, and links to code repositories or live demos. Use AI to suggest descriptions and improvements of Rohit's existing projects.
- Skills and Expertise: Visually represent Rohit's skills and expertise in data analysis, machine learning, and related tools.
- Contact Form: Allow visitors to contact Rohit Mirge through a simple form.
- Resume/CV: Provide an option to download Rohit's resume/CV in PDF format.
- Blog Posts/Articles: Optionally include a blog or articles section to showcase thought leadership, leveraging AI as a tool to suggest topic ideas and titles.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), to convey intelligence, insight, and data depth.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop for content and visual elements.
- Accent color: Teal (#008080), highlighting calls to action and interactive elements with a sense of calm professionalism.
- Body font: 'Inter', sans-serif, for clean, readable text throughout the website.
- Headline font: 'Space Grotesk', sans-serif, for prominent titles and headings with a modern, data-driven feel.
- Use minimalist, professional icons to represent skills, tools, and data concepts.
- Implement a clean, grid-based layout with clear visual hierarchy to present information effectively.
- Use subtle animations and transitions to enhance user experience without being distracting.
